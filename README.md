# Overview

This project is designed to help me strengthen my skills as a software engineer by practicing core programming concepts in JavaScript. The goal is to build an interactive web application, apply modern JavaScript features, and document the development process.  

The software is a web-based To-Do List application that runs in the browser and demonstrates dynamic user interaction. The application allows users to add tasks, edit tasks inline, mark tasks as completed, delete tasks, filter tasks, and manage task categories. It also includes timestamps using an external library and stores data using localStorage. This demonstrates understanding of JavaScript syntax, DOM manipulation, event handling, and state management.  

The purpose of creating this software is to build a strong foundation in front-end development, practice writing clean and maintainable code, apply problem-solving skills, and prepare for more advanced full-stack projects in future modules.

# Features Demonstrated

* Variables, Expressions, and Conditionals  
* Loops and Array Methods (`map()`, `filter()`, `forEach()`)  
* Functions for application features such as `addTask()`, `editTask()`, `deleteTask()`, and `toggleTask()`  
* Recursion (`countdownTasks()` function)  
* DOM Manipulation (creating and updating elements dynamically)  
* Event Handling (button clicks and user input)  
* Local Storage (saving and retrieving tasks)  
* External JavaScript Library (Day.js for timestamps)  
* Exception Handling (`try...catch` for validation and errors)  

# How to Run

1. Open the project folder (`ToDoApp`) on your computer.  
2. Locate the `index.html` file.  
3. Double-click the file or open it in a web browser (Chrome, Edge, or Firefox).  
4. Start adding and managing tasks directly in the browser.
